# Crypto Social Sentiment Analyzer (CSSA) - Windows Installer
CSSA is a desktop application for Windows that analyzes sentiment from social media and news about cryptocurrencies and provides data for automated trading. The software leverages aggregated data from sentiment-tracking platforms to monitor market mood and identify emerging trends.

Status: Windows installer is ready for use. Version 2.5.2 is available.

# Key Features
Sentiment Analysis: Retrieve the sentiment balance (positive vs. negative mentions) for specific cryptocurrencies.

Social Volume Tracking: Monitor the total number of social media mentions and detect significant shifts (spikes or drops).

Social Dominance: Measure an asset's share of discussion within the overall crypto media landscape.

Trending Words: Identify the most popular terms currently trending in cryptocurrency discussions.

# Getting Started
The installation is straightforward via the provided setup.exe installer for Windows.

# Installation Steps:

Download the latest Setup-v2.5.2.exe file from the Releases page.

Run the Setup-v2.5.2.exe file as an administrator.

Follow the on-screen instructions in the installation wizard.

Launch Crypto Social Sentiment Analyzer from your Start Menu or Desktop shortcut after installation.

# Prerequisites
Operating System: Windows 10 or 11 (64-bit).

# Configuration
Upon first launch, the application will guide you through the initial setup, where you can enter your API key and configure basic settings.

# Examples
Once running, you can use the application to query data. Here are some examples of the insights you can get:

Input Query: "What's the current sentiment for Bitcoin?"

Output: "Bitcoin's sentiment balance over the past 24 hours is +15.2."

Input Query: "Show me the social volume trend for Ethereum this week."

Output: "Ethereum's social mentions have increased by 45% in the last 7 days."

Input Query: "List the top 5 trending crypto words."

Output: "Top trending terms: 'halving', 'ETF', 'layer2', 'memecoin', 'regulation'."
